# myOthello-gui-2
myOthello-gui ver.2

オセロを作ってみました。

人間(黒) 対 ボット(白) です。

ボットの強さは、気を抜いてやると負けるぐらいにはしてあります。

confフォルダに設定ファイル (game.conf) をおいています。

そこで盤面の数を調整できます。

picフォルダにマス画像を置いています。

# 遊び方

binフォルダにて

> java Main

jarファイルを作成したなら、(myOthello.jar という名前とする）

myOthelloフォルダにそのjarファイルを置くとすると、

そのフォルダの中に、picフォルダとconfフォルダを置きます。

> java -jar myOthello

これで遊べます。
